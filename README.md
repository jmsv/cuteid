# cuteid
cute id generator

[![Build Status](https://travis-ci.org/jmsv/cuteid.svg?branch=master)](https://travis-ci.org/jmsv/cuteid)

## Usage

```js
const cuteid = require('cuteid').generate

const id = cuteid()
console.log('id:', id) // tiny-lizards (or something)
```
